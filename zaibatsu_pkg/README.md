# 財閥 — 明治维新·财阀 对局助手

一个**离线、零网络依赖**的桌面小程序，根据《Nippon Zaibatsu 对局助手 设计文档（最终版）》实现，
帮助你在对局中快速录入棋局状态并获得**行动推荐 + 理由解释**。

打包后是一个**单文件 `.exe`**，双击即用，无需安装任何运行环境。

---

## 一、目录结构

```
zaibatsu/
├─ app/                  ← 应用本体（前端，全部逻辑都在这里）
│  ├─ index.html         ← 界面（三步开局向导 + 双栏对局界面 + 推荐抽屉）
│  ├─ config.js          ← 规则与配置：§3 60 张工厂效果、影响力表、人数配置等
│  ├─ engine.js          ← 决策引擎：走子生成 → 模拟 → 评估 V(state) → 搜索 → 解释
│  └─ app.js             ← 界面控制器：录入、渲染、撤销、存档/读档
├─ main.py               ← 启动器（.exe 外壳）：内置本地服务 + 原生窗口
├─ app.spec              ← PyInstaller 打包配置
├─ build_exe.bat         ← Windows 一键打包脚本
├─ requirements.txt      ← 打包/运行依赖（仅外壳需要）
├─ app_icon.ico          ← 应用图标（朱印红「財」印）
└─ README.md             ← 本文件
```

> 说明：**应用的全部功能都在 `app/` 的前端代码里**，`main.py` 只是负责把它装进一个原生窗口。
> 这样打包最稳、离线运行，也方便你直接用浏览器打开 `app/index.html` 调试。

## 二、在 Windows 上生成 .exe（推荐）

> 本项目在 Linux 容器中开发，**无法直接编译出 Windows 可执行文件**，
> 因此交付完整可运行工程 + 打包脚本，请在 Windows 上一键生成 `.exe`。

1. 安装 [Python 3.9+](https://www.python.org/downloads/)（安装时勾选 *Add Python to PATH*）。
2. 把整个 `zaibatsu/` 文件夹拷到 Windows 上。
3. **双击 `build_exe.bat`**，脚本会自动：
   - 安装依赖 `pyinstaller`、`pywebview`；
   - 清理旧产物；
   - 打包成单文件。
4. 完成后在 `dist\财阀对局助手.exe` 即得到可执行文件，双击运行。

手动命令（等价于脚本）：

```bat
pip install pyinstaller pywebview
pyinstaller app.spec
```

## 三、免打包，直接运行（任意系统）

不想打包也能用，两种方式：

**A. 直接用浏览器打开**
双击 `app/index.html` 即可（Chrome / Edge 等现代浏览器）。

**B. 用启动器运行（带原生窗口）**

## 四、怎么用

1. **开局三步**：选人数（3/4 人）与座位 → 选风格偏好（均衡 / 胜率优先 / 分数优先）→ 开始对局。
2. **录入状态**：左栏是公共版图（进度、雇工区、工人队列、地区、工厂池），
   右栏是各玩家面板（用 +/− 步进器和增删列表维护工厂、工人、合同、好感）。
3. **求推荐**：点底部「**求最优解**」，右侧抽屉给出
   - 盖着朱印的**首选行动卡**；
   - **推荐理由**（来自评估函数各项贡献）；
   - **备选方案 Top-3** 及其评分。
4. 还支持：记录对手回合、回合重置、撤销、存档/读档（JSON）。

## 五、关于游戏数据（重要）

设计文档 §9 列出了一批**需要对照实体卡牌校验**的数值（部分合同、起始代币 B 值、
回合轨胜分、地区名/海外公司值、岩仓使节团 6 个目的地、后期工厂模块 #5、时代排名分等）。

这些值在 `app/config.js` 的 `TODO_DATA` 中以**占位形式**标注（`todo:true`），
**不影响引擎运行**——你可以在对局面板里据实填写/调整。校对实体卡后，
直接在 `config.js` 中替换占位值即可，无需改动引擎。

## 六、技术要点

- **引擎（已按官方规则书/玩家手册校准）**：
  - **取胜主路径——地区影响力名次分**：逐地区计算「我方影响力 + 火车(每列+2，需该区有我标记) vs 各对手 vs 海外公司」的排名，按规则书三时期分值 10/8/6/2 → 18/14/10/3 → 26/20/14/4 给分，并对剩余时期加权。本地市场/火车类推荐会直接给出「该区名次从第几名升到第几名、还有几次结算」的解释。
  - **搜索**：单步贪心打分 + 对头部候选做 2 步前瞻精排（缓解短视，提升取胜质量）。
  - **评估** `V(state)` 综合：地区名次分、三轨终局个人目标(5/8/10→10/16/20 阈值)、引擎产能、当前分、资源/合约折现、工人节奏与整顿成本，并按偏好（胜率/分数）动态调权。
  - **配置**：§3 全部 60 张工厂效果用 EffectSpec 结构编码，模拟器统一执行；部门奖励、影响力面值表(1级1/2/3、2级3/4/5、3级5/6/7)、岩仓 6 目的地、影响力标记(1–7)均按规则书录入。
- **离线**：仅用系统字体，无任何在线资源；`.exe` 双击即用。

## 七、界面视觉（严格对照盘面布局重做）

> 出于版权考虑，软件**不嵌入规则书/手册中的实际图片**，而是用 CSS/SVG **还原同款盘面布局、配色与图标排布**，不复制受版权保护的美术素材。

- **海图视图（对照 Common Setup 盘面）**：主图版做成一张「海图」——海蓝渐变 + 列岛剪影，顶部覆盖**雇用区/行动条**（上工人槛、下行动图标）与左上**工人列候补**；6 个**市场菱形**按盘面走向斜向散布在海图上，每个旁边带**灰色海外公司丸数字**；右下角覆盖**岩仓使节团 2×3 表**。点击任一菱形即在下方展开该市场的编辑面板。
- **市场可完全自定义**：每个市场关联一张**需求板**（两类货物 + 朝向），点击货物徽标循环切换种类、点「翻转朝向」改变正反面；据此自动决定该市场接受的货物等级与影响力面值（L1:1/2/3、L2:3/4/5、L3:5/6/7）。面板内含 4 个本地市场格（点选占用方）与「我方/火车/海外/各对手」影响力录入，并实时显示当前名次与本时期名次分。
- **财阀板（对照 Individual Setup 盘面）**：右栏玩家板做成「财阀板」——左部门列（6 色，点击循环 未获得→基础→升级）、中三轨阶梯（12 格 + 5/8/10 阈值）与计分轨、右船/设备/火车阶梯、底部工厂横排（建筑卡片 + 货物/设备）、个人目标(恩赐)图标槛。
- **契约卡片墙（对照 Contracts 区）**：8 张固定契约渲染成紫框卡片 + 部门色图标，完成时右上角弹绿色对勾，点击切换。
- **起始标记卡（对照 Starting Conditions）**：开局展示 5 张可点选标记卡，选中自动计入起始分。

## 八、扩展模组（严格遵照规则书）

- **人数 2/3/4 + 单人**：按规则书设定入袋工人 36/42/48、工人列 1/2/3 行、市场 2/3/4 格、船位 2/3/4、火车位 3/5/7；影响力名次 2 人局跳过第 2 档。
- **后期工厂模组（Late-Game Factories）**：开局向导可勾选；勾选后工厂池追加各类型「编号 5」工厂（规则书 setup 4b：2/3/4 人加 3/4/5 张），全部 6 种编号 5 效果已按手册编码。
- **单人模式（Automa）**：可勾选，自动切到 2 人局规则；Automa 仅执行火车/本地市场行动。引擎推荐始终只为「我」求解。
